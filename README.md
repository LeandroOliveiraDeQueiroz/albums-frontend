# albums-frontend

**Obs.: All development was made using Node v20.9.0**

## Run <a name="run"></a>

**Requirements:**

- npm,

**Install:**

1.  Clone the repository:

  ```bash
  git clone https://github.com/LeandroOliveiraDeQueiroz/albums-frontend.git
  cd albums-frontend
  ```
3.  Install node modules:

  ```
  npm install
  ```

**Exec.:**

1.  To run the project in development:

  ```bash
  npm run dev
  ```
